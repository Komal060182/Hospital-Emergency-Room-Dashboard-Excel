# Hospital-Emergency-Room-Dashboard-Excel
Interactive Hospital Emergency Room Dashboard created using Microsoft Excel
## 📊 Project Overview
This interactive dashboard is created using Microsoft Excel to analyze hospital emergency room data and generate meaningful insights for better decision-making.

## 🔎 Key Insights
- Patient Admission Status (Admitted vs Not Admitted)
- Patient Age Distribution
- Patients attended within 30 minutes
- Gender-wise Analysis
- Department Referrals

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Visualization
- Data Cleaning
